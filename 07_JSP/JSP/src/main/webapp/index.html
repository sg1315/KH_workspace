JSP

JSP란, HTML내에서 자바코드를 쓸 수 있는 "자바언어"로
Servlet에서 응답화면(html)을 구현했던 복잡함을 간단하게 하는 기능
=> 즉, JSP의 가장 큰 장점은 Servlet.에서 비즈니스로직에만 직중하고
View를 담당하는 코드를 분리해서 JSP에서 작성할 수 있다는 것
(JSP도 내부적으로는 Servlet으로 동작함)
내장 객체 제공(request, response, session등...)

JSP Element표현법

1. JSP스크립트 원소

JSP페이지에서 잡바코드를 기술할 수 있는 기능

  1. 스크립틀릿 : <% 자바코드 %>
    JSP에세 자바코드를 기술하기위한 제일 기본적인 공간
  2. 표현식(출력식) : <%= 자바변수 %>
    자바에서의 값을 화면상에 출력하기위해 사용

2. 지시어 <%@ page | include | taglib 속성=값 속성=값 ... %>

JSP페이지 전체에 영향을 미치는 정보를 기술할 때 사용

  1. page지시어 : 현재 jsp페이지를 처리하는데 필요한 각종 속성들을 기술하는 부분
    • language: 사용할 언어 유형
    • contentType: 해당 페이지의 형식, 문자셋 지정
    • pageEncoding: JSP페이지내에 기록된 자바코드의 인코딩 방식
    • import: 자바의import와 같은 의미
    • errorPage: 해당jsp상에 오류가 발생했을 경우 보여줄 에러페이지를 미리 기술
  2. include 지시어 : 현재 jsp상에 또다른 jsp를 포함시켜 사용하는 지시어
  3. taglib 지시어 : jsp기능을 좀더 확장시킬 수 있는 추가적인 라이브러리를 등록하는 지시어

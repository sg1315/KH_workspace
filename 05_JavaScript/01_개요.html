자바스크립트 개요

1. 스크립트 언어란?

프로그램의 동작을 사용자의 요구의 맞게 수행되도록 해주는 용도
빠르게 배울 수 있고 짧은 소스코드로 상호작용이 되도록 구성되어 있음

2. 자바스크립트

웹 브라우저에서 많이 사용되는 인터프리터방식의 스트립트 언어
- 인터프리터방식 : 코드를 한줄씩한줄씩 읽어가며 바로 실행하는 방식
(실시간으로 텍스트 분석해서 실행됨 -> 코드에 문법 오류가 있다면 런타임 시점에 알려줌)

3. 자바스크립트 사용법

  1. inline(인라인) : 태그내에서 간단한 소스코드를 작성해서 실행되게 하는 방법
  2. internal(내부) : html문서 내에 소스코드를 작성해서 실행되게 하는 방법
  3. external(외부) : 별도의 .js파일로 소스코드를 작성해서 가져다가 실행되게 하는 방법

1) inline

[표현법] (태그 on이벤트명="해당 요소의 이벤트 발생시 실행할 코드")



2) internal방식

script태그 영역안에 소스코드를 작성
script태그는 head태그, body태그 다 작성 가능

3) external방식

외부의 .js파일을 생성해서 따로 js코드를 작성한다.

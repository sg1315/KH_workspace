selectbox

무조건 하나를 선택해야 할때

radio button

selectBox처럼 여러개 중에 하나를 선택할 때 주로 사용
name속성을 통해서 여러개의 라디오버튼을 하나의 선택지로 묶을 수 있음
checked속성을 넣은 라디오버튼이 기본값으로 설정됨
id - for 속성을 통해서 input과 label태그를 묶을 수 있음
서버에 실제로 전달될 때 key - name속성값, value -> value속성값으로 전달
